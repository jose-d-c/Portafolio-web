# Checks de consistencia

- Cada `slug` debe tener `getStaticPaths` alcanzable
- Cada `rutaCodigo` o `rutaCodigoPlantilla` debe existir en `public/`
- Cada `rutaImagen` o `rutaImagenPlantilla` debe existir en `public/`
- `enlaceExterno` debe coincidir con la ruta publica real
- Titulos y descripciones deben corresponder al ejercicio real
