# Calidad de contenido

- Evitar placeholders si el pedido apunta a contenido final.
- Usar lenguaje profesional, concreto y verificable.
- Mantener consistencia entre titulos, descripciones y etiquetas.
- Preferir descripciones breves que expliquen valor y tecnologia.
- Si el contenido cambia tono o nivel de madurez, revisar meta description del layout.
